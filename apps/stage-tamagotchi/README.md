# たまごっち アイリ

A desktop application for たまごっち アイリ.

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ cd /packages/stage
$ pnpm dev:tamagotchi
```

Then open another terminal and run:

```bash
$ cd /packages/tamagotchi
$ pnpm dev:tamagotchi
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
