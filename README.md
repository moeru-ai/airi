## ✨ Core Features

### 1. Deep AIRI Core Integration
- **Shared Context**: Seamlessly connects with the native AIRI system for cross-platform and multi-terminal memory synchronization.
- **Long-term Memory System**: The bot remembers your preferences, habits, and past conversation details—no more "ephemeral" interactions.

### 2. Flexible Interaction Experience
- **Full Format Support**: Perfectly integrates with the QQ messaging protocol, supporting text, emojis, and rich media.
- **Optional Voice Reply**: Configurable triggers allow the AI to communicate via voice (supports TTS switching).

---

## 🛠️ Current Features

- [x] **Basic Communication**: Stable real-time sending, receiving, and parsing of QQ messages.
- [x] **Memory Awakening**: Messages sent via QQ are automatically linked to the AIRI memory bank (in-context).
- [x] **Speech Synthesis**: Option to return results as voice messages during conversations.
- [x] **Multi-turn Dialogue**: Precise context maintenance for logically coherent, deep conversations.

---

## 📅 Roadmap

- [ ] **Multimodal Emotional Expression**: Recognize emotions based on semantics and randomly send matching stickers/emojis.
- [ ] **Proactive Socializing**: Moving beyond passive commands; the AI will initiate chats based on time, weather, or memories.
- [ ] **Full-Duplex Voice Interaction**:
    - Integrate **OpenAI Whisper** and other models to parse incoming QQ voice messages.
    - Achieve a "Voice-In, Voice-Out" natural communication loop.
- [ ] **Intelligent Personality Evolution**: Dynamically adjust favorability levels and reply styles based on interaction frequency.

## ✨ 核心特性

### 1. 深度整合 AIRI 核心
- **共享上下文**：与 AIRI 原生系统无缝对接，实现跨平台/多端记忆同步。
- **长效记忆系统**：机器人能记住你的喜好、习惯及过往对话细节，不再是“阅后即焚”。

### 2. 灵活的交互体验
- **全格式收发**：完美对接 QQ 消息协议，支持文字、表情及富媒体。
- **可选语音回复**：可配置触发开关，让 AI 用声音与你交流（支持 TTS 切换）。

---

## 🛠️ 已实现功能 (Current)

- [x] **基础通讯**：稳定支持 QQ 消息的实时收发与解析。
- [x] **记忆唤醒**：QQ 发送的消息会自动关联至 AIRI 的记忆库（上下文中）。
- [x] **语音合成回复**：对话时可选择以语音消息形式返回结果。
- [x] **多轮对话**：精准的上下文维持，支持逻辑连贯的深度长聊。

---

## 📅 未来路线图 (Roadmap)

- [ ] **多模态情感表达**：根据语义识别情感，随机发送匹配的表情包。
- [ ] **拟人化主动社交**：不再被动等待指令，AI 将根据时间、天气或记忆主动找你聊天。
- [ ] **全双工语音交互**：
    - 集成 **OpenAI Whisper** 等模型解析 QQ 语音消息。
    - 实现“语音进-语音出”的纯自然交流闭环。
- [ ] **智能人格演变**：根据互动频率动态调整好感度与回复风格。
