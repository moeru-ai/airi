# AI カバー（歌声変換）

AIRI の歌唱モジュールは、任意の楽曲を指定キャラクターの声に変換します。

## 機能概要

- **キャラクターカバー**: 事前学習済み RVC モデルで楽曲を変換
- **ゼロショットカバー**: 参照音声をアップロードするだけで声をクローン（Seed-VC）
- **自動パイプライン**: ボーカル分離 → ピッチ抽出 → 音色変換 → ミキシング
- **成果物トラッキング**: 各ジョブで manifest.json の監査記録を生成

## 技術スタック

| ステージ | 技術 |
|---------|------|
| ボーカル分離 | MelBand-RoFormer / BS-RoFormer |
| ピッチ抽出 | RMVPE |
| 音色変換 | RVC（メイン）/ Seed-VC（ゼロショット） |
| 後処理 | FFmpeg |
