## まずはチャットを開始してください

AIRI をインストールして起動した後、最初の起動に従って基本設定を完了できます。

1. AIRI が言語の選択を求めてきた場合は、まず使用する言語を選択します。
2. [**独自の AI サービス ソースを構成する**] を選択します。すでに AIRI アカウントを使用している場合は、ログインすることも選択できます。
3. OpenRouter、OpenAI 互換 API、DeepSeek、Ollama、Qwen、Gemini、Claude などのチャット サービス ソースを選択します。
4. API キー、ローカル サービス アドレス、その他の必要な情報を入力します。
5. チャット モデルを選択し、保存して続行します。
6. メイン キャラクター ウィンドウに戻ったら、右下隅のコントロール アイランドにある [**展開**] ボタンをクリックします。
7. [**チャットを開く**] をクリックし、メッセージを入力して送信します。

::: tip Ollama をローカルで使用しますか?
システム環境変数 `OLLAMA_ORIGINS=*` を設定し、Ollama を再起動して AIRI から選択する必要があります。
:::

## 何が見えるでしょうか

デスクトップ版はステージたまごっちとも呼ばれ、通常は次のインターフェイスで構成されます。

- **メイン キャラクター ウィンドウ**: Live2D/VRM ステージはデスクトップ上に常駐します。
- **コントロール アイランド**: メイン キャラクター ウィンドウの右下隅にある小さなボタンのグループ。
- **チャット ウィンドウ**: コントロール アイランドから開かれる会話ウィンドウ。
- **設定ウィンドウ**: サービス ソース、役割、モデル、本体モジュール、データ、接続、およびシステム オプションを構成します。
- **システム トレイ メニュー**: サイズ変更、位置調整、設定、字幕、ウィジェットを開いて終了します。

メイン キャラクター ウィンドウが非表示になっている場合は、AIRI トレイ アイコンをクリックするか、トレイ メニューで **表示** を選択すると、メイン キャラクター ウィンドウを元に戻すことができます。

## コントロールアイランド

コントロール アイランドは、デスクトップ バージョンの日常操作に最も便利なエントリ ポイントです。

- [**展開**] をクリックして、さらにアクションを表示します。
- [**チャットを開く**] をクリックしてチャット ウィンドウを開きます。
- [**設定を開く**] をクリックして、サービス ソース、モデル、機体モジュール、役割、およびシステム設定を構成します。
- [**役割の切り替え**] をクリックして、現在のキャラクター カードを変更します。
- 必要に応じて、[**更新**] をクリックしてステージをリロードできます。
- テーマを切り替えるには、明るい/暗いアイコンをクリックします。
- 画鋲アイコンをクリックすると、ウィンドウが最上部に切り替わります。
- 目のアイコンをクリックして、**ホバー時に非表示** / **常に表示** を切り替えます。
- マイク ボタンをクリックして聴覚コントロールをオンにします。
- 移動ボタンをドラッグしてメインキャラクターウィンドウを移動します。

## ホバー時に非表示にする

目のアイコンは、AIRI の表示方法を切り替えるために使用されます。つまり、クリック可能な状態を維持するか、作業中の遮蔽やクリックの邪魔を最小限に抑えます。

- **常に表示** では、キャラクターが表示され、クリック可能な状態が維持されます。
- **Hide on Hover** は、カーソルが近づくとキャラクターとインターフェイスをフェードアウトし、下のアプリケーションをクリックしやすくします。

ホバー時の非表示を初めてオンにすると、AIRI は短い説明をポップアップ表示します。オンにした後で AIRI をクリックするのが不便な場合は、カーソルをコントロール アイランドの近くに移動し、目のアイコンをもう一度クリックして元に戻します。

## 移動とサイズ変更

メイン キャラクター ウィンドウを移動するには、コントロール アイランドの右下隅にある移動ボタンをドラッグします。

Windows では、ウィンドウの端または隅をドラッグしてサイズを変更できます。トレイ メニューには、いくつかの一般的なサイズも用意されています。

1. AIRI トレイ アイコンを右クリックします。
2. **サイズ変更**を開きます。
3. **推奨**、**全高**、**半高**、または**全画面**を選択します。

同じトレイメニューの**整列**を使用すると、ウィンドウを画面の中央または四隅に移動できます。

## これらの設定をもう一度確認することをお勧めします

最初のチャットを完了したら、次のページに目を通すことをお勧めします。

- **サービス ソース**: チャット、スピーチ、文字起こし、アーティストのサービス ソースを追加または編集します。
- **Body Module**: 意識、発声、聴覚、視覚、記憶、Discord、Minecraft、Factorio、MCP およびその他のモジュールのサービスを選択します。
- **キャラクター モデル**: Live2D/VRM モデルを切り替えるか、独自のモデルをインポートします。
- **AIRI キャラクターカード**: 現在のキャラクターを切り替えるか、新しいキャラクターカードを作成します。
- **システム**: 言語、テーマ、データ分析設定、デスクトップ固有のオプションを設定します。

一部のモジュールはまだ実験段階にあり、ローカルのソース コード構成や追加の外部サービスが必要になる場合があります。 Windows の使用方法の詳細については、[デスクトップ バージョンの詳細な説明](./setup-and-use/) を参照してください。
