---
title: Web検索を設定する
description: Tavily を使用して AIRI が最新の会話を検索できるようにする
---

AIRI は、最新の資料や馴染みのない情報が必要なときに、Web 検索を使用してインターネットにクエリを実行できます。独自の Tavily API キーを使用します。有効にすると、AIRI は適切な会話で検索ツール自体を呼び出し、回答に使用される実際のソース リンクを含めます。

## 前提条件

- AIRIがインストールされ、起動されます。
- すでに Tavily アカウントをお持ちで、[Tavily](https://tavily.com/) コンソール) から API キーを作成します。
- ツールを使用して呼び出すことができるチャット プロバイダーとモデルを構成しました。 AIRIがツールを使用できない場合は、まずツール呼び出しに対応した機種に変更してください。

::: 警告 API キーのセキュリティ
Tavily API キーは、現在のデバイスにのみ保存する必要があります。リポジトリに送信したり、他の人に送信したり、キャラクター カード、日記、スクリーンショットに含めたりしないでください。キーが侵害された疑いがある場合は、すぐにキーを取り消して、Tavily コンソールで新しいキーを作成してください。
:::

## 構成手順

1. **[設定] → [本体モジュール] → [ネットワーク検索]** を開きます。
2.「Web検索を有効にする」をオンにします。
3. 「Tavily API Key」にAPIキーを貼り付けます。
4. 「Web 検索の準備ができました」と表示されたら、チャットに戻ることができます。設定は自動的に保存されるため、保存ボタンをクリックする必要はありません。

スイッチをオフにするか API キーをクリアすると、AIRI は Tavily に検索リクエストを送信しなくなります。

## AIRIはいつ検索しますか?

AIRI は既存の知識を優先します。 Web 検索は、明示的に検索を要求した場合、または質問に急速に変化する情報が含まれる場合にのみ使用されます。ニュース、価格、最近のリリース、現在のアクティビティ、ライブチャート、最新のドキュメントなど。

より正確に検索したい場合は、次のように目標と範囲を直接指定してください。

- 「AIRI の最新の安定バージョンのリリース ノートを検索し、リンクを添付してください。」
- "Tavily の公式ドキュメントで API キーに関する手順を見つけてください。"
- 「`github.com/moeru-ai/airi` の最新週のアップデートのみを検索します。」

検索結果にはソースへのリンクが含まれます。 AIRI は、実際にクエリされたリンクのみを引用できます。答えで十分な情報が見つからない場合は、検索を続行するか、不確実性を明確に述べる必要があります。

## プライバシー、信頼性、セキュリティ

検索ごとにクエリ テキストが Tavily に送信されます。したがって、API キー、パスワード、アクセス トークン、プライベート アドレス、または第三者に提供すべきではないその他の情報を検索用語に含めないでください。検索結果には、不正確なコンテンツ、古いコンテンツ、または偏ったコンテンツが含まれる場合もあります。

::: 警告 重要な情報をご確認ください
検索結果は AIRI の参照のみを目的としており、元の質問や運用目標が自動的に変更されることはありません。アカウント、セキュリティ、医療、法律、または財務問題に関連するコンテンツについては、ソースのリンクを開いてご自身で確認し、公式または一次ソースを優先してください。
:::

＃＃ よくある質問

### 設定済みを表示しますが、AIRI は検索しません

まず、ネットワーク検索スイッチがまだオンになっていることを確認し、現在のチャット モデルがツール呼び出しをサポートしていることを確認します。次に、チャットで「ソースリンクを検索して添付してください」と直接リクエストします。それでも呼び出されない場合は、モデル サービス プロバイダーがツール呼び出しリクエストを許可しているかどうかを確認してください。

### プロンプト API キー、権限、または割り当てエラー

Tavily コンソールに戻って、キーが完全でまだ有効であることを確認し、アカウントの利用可能なクレジットまたはアクセス権を確認します。コピーするときは、先頭または末尾のスペースや改行を含めないでください。キーを変更した後、AIRI に戻って再度貼り付けるだけです。

### 検索結果が不正確であるか、最新ではありません

質問には、「過去 1 週間のみ確認する」「公式文書のみを使用する」など、使用する期間、場所、ソースを明記してください。添付のリンクを開いて重要な結論を確認してください。インターネット検索は、専門家のアドバイスや独立した判断に代わるものではありません。
