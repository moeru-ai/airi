---
title: BytePlus Coding Plan
description: AIRI での BytePlus コーディング プラン チャット モデルの設定
---

BytePlus コーディング プランは、AIRI で独立したプロバイダー カードとして利用できます。

::: info BytePlus コーディング プランを選択する理由?
BytePlus アカウントにコーディング プランがある場合は、対応するサービス プランに一致させるために、通常の BytePlus 構成の代わりにこのカードを使用する必要があります。
:::

## ステップ 1: BytePlus コーディング プランの認証情報を準備する

1. [BytePlus コンソール](https://console.byteplus.com/)，在 コーディング プランの対応ページ)を開いてログインし、認証情報とエンドポイント情報を取得します。

::: 警告 API キーのセキュリティ
API キーやエンドポイント認証情報をリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [BytePlus コーディング プラン]** を開き、BytePlus コンソールをクリックして、プランの資格情報とエンドポイント情報を入力します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークと資格情報が正しいかどうかをテストします。
2. **モデルの選択**: テストが成功したら、**設定 → 認識**で利用可能なモデルを選択します。

## トラブルシューティング

検証が失敗した場合は、資格情報とエンドポイントが同じ BytePlus コーディング プランに属しているかどうかを確認します。モデルをロードできない場合は、プランがターゲット モデルへのアクセスを許可していることを確認してください。
