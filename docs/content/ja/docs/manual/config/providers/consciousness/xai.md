---
title: xAI
description: AIRI での xAI Grok チャット モデルの設定
---

xAI プロバイダーにより、AIRI は Grok チャット モデルを使用できるようになります。設定方法は一般的なAPI Keyサービスプロバイダーと同様です。

::: info なぜ xAI を選ぶのですか?
すでに xAI API アカウントをお持ちで、AIRI で Grok モデルを使用したい場合は、このサービス プロバイダーを選択できます。
:::

## ステップ 1: API キーを作成する

1. [xAI Developer Console](https://console.x.ai/) を開いてログインし、必要な API キーまたは認証情報を作成します。
2. アカウントが API 使用のためにアクティブ化されており、使用可能な割り当てがあることを確認します。
3. キーをコピーします。

::: warning API キーのセキュリティ
API キーはパスワード マネージャーまたは AIRI のローカル設定にのみ保存してください。キーをコードに書き込んだり、リポジトリにコミットしたり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [xAI]** を開きます。
2. API キーを入力します。
3. デフォルトのベース URL `https://api.x.ai/v1/` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

Ping API が失敗した場合は、まず API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストが利用できない場合は、xAI ドキュメントに記載されているモデル ID を [認識] ページに手動で入力できます。
