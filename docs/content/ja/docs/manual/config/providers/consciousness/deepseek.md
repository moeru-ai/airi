DeepSeek は、OpenAI 形式と互換性のあるチャット API を提供します。このページでの設定を完了すると、AIRI は、Awareness の DeepSeek によって提供されるモデルを使用できるようになります。

::: info DeepSeekを選ぶ理由?
AIRI で DeepSeek モデルを使用したい場合、またはすでに DeepSeek API キーを持っている場合は、このサービス プロバイダーを直接選択できます。
:::

## ステップ 1: API キーを取得する

1. [DeepSeek管理コンソール](https://platform.deepseek.com/)。)を開く
2. [API キー] ページで新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害された場合は、直ちにそのキーを取り消し、DeepSeek コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [DeepSeek]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.deepseek.com/v1` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、DeepSeek によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
