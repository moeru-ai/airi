Zhipu AIは、OpenAI形式と互換性のあるチャットAPIを提供します。このページでの設定を完了すると、AIRIは「Consciousness」でWisdom AIが提供するモデルを利用できるようになります。

::: info Zhipu AIを選ぶ理由?
AIRI で Zhipu AI モデルを使用したい場合、またはその API キーをすでに持っている場合は、このサービス プロバイダーを直接選択できます。
:::

## ステップ 1: API キーを取得する

1. [Zhipu AI API キー](https://open.bigmodel.cn/usercenter/apikeys)。)を開く
2. 新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害されたら、すぐにそれを取り消し、Zhipu AI コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Zhipu AI]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://open.bigmodel.cn/api/paas/v4/` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストを読み込めない場合は、Wisdom AI が提供する正確なモデル ID を [認識] ページに手動で入力できます。
