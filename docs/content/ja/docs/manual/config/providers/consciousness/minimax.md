このページは、中国本土の MiniMax オープン プラットフォームで作成された API キーに適用されます。 MiniMax は、OpenAI 形式と互換性のあるチャット API を提供します。設定が完了すると、AIRI はそのモデルを 意識 で使用できるようになります。

::: info MiniMax を選ぶ理由
中国本土の MiniMax オープン プラットフォームで API キーを作成する場合は、このサービス プロバイダーを選択する必要があります。海外プラットフォームで作成されたキーには[MiniMax Global](./minimax-global.md)をご利用ください。
:::

## ステップ 1: API キーを取得する

1. [MiniMaxコンソール](https://platform.minimaxi.com/)。)を開きます
2. [API キー] ページで新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害された場合は、直ちにそのキーを取り消し、MiniMax コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [MiniMax]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.minimaxi.com/v1/` をそのまま使用します。中国本土と海外のプラットフォームの API キー、請求書、ベース URL を混在させることはできません。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、MiniMax によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
