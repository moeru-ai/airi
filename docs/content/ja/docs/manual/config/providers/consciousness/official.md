---
title: AIRI公式プロバイダー
description: AIRI公式をチャットプロバイダーとして使用する
---

AIRI をすぐに使い始めて設定を簡素化したい場合は、公式プロバイダーを使用してチャットを有効にすることができます。

::: info なぜAIRI公式プロバイダーを選ぶのですか?
サードパーティ API キーを自分で設定せずに AIRI をすぐに体験したい場合は、公式プロバイダーを選択できます。
:::

## ステップ 1: アカウントにログインします

1. ログイン ボタンをクリックすると、ブラウザ ページが開きます。
2. ログイン方法を選択します。現在、電子メール、Google、GitHub ログインをサポートしています。

    ::: info 中国人ユーザーへの特別な注意事項
中国のユーザーは、Google および GitHub にログインできない場合があります。ログインにはメールアドレスを使用してください。携帯電話番号での登録とログインは現在サポートされていません。
    :::

3. プロンプトに従ってログイン/登録を完了します。

    ::: info
メールが届かない場合は、スパムメールに入っていないか確認してください。
    :::

## ステップ 2: 利用可能な電力を確認する

新しいアカウントにはそれぞれ 150 の権限があります。会話ごとにバッテリーの一部が消費されます。電池が消耗すると通話を継続できなくなります。

## ステップ 3: バッテリーを充電する

現在、リチャージは AIRI Web バージョンでのみサポートされています。ウェブ版にログインした後、**設定 → Flux** を開き、バッテリー パックを選択し、ページの指示に従って支払いを完了します。デスクトップ版は現在直接充電をサポートしていません。

## トラブルシューティング

ログインメールが届かない場合は、迷惑メールフォルダをご確認ください。利用可能なバッテリー パックが Web ページに表示されない場合、または支払いを作成できない場合は、しばらくしてからもう一度お試しください。デスクトップで充電する必要がある場合は、AIRI Web バージョンにアクセスして操作を完了してください。
