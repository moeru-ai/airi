---
title: Azure OpenAI
description: AIRI で Azure OpenAI チャット モデルを構成する
---

Azure OpenAI は、Azure リソース エンドポイントとデプロイを通じてモデルにアクセスします。

::: info Azure OpenAI を選ぶ理由?
チームがすでに Azure OpenAI でモデルをデプロイし、アクセス許可を管理している場合、これはオンボードする簡単な方法です。
:::

## ステップ 1: Azure OpenAI リソースを準備する

1. [Azure Portal](https://portal.azure.com/)，创建或打开 Azure OpenAI リソース)を開いてログインし、エンドポイントと API キーを取得します。

::: warning API キーのセキュリティ
Azure API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Azure OpenAI]** を開き、**API キー** と Azure OpenAI エンドポイントを入力します。
2. コンソールから提供される完全なチャット完了アドレスを入力することをお勧めします。アドレスに展開名と `api-version` が含まれている場合、AIRI はそれに応じて設定を識別します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワーク、エンドポイント、および API キーが正しいかどうかをテストします。
2. **モデルの選択**: テストが成功したら、対応する展開を選択し、**設定 → 認識** に移動して有効にします。

## トラブルシューティング

検証が失敗した場合は、API キー、エンドポイント、デプロイ名、`api-version` がすべて同じ Azure OpenAI リソースからのものであることを確認してください。プレゼンテーション専用のモデル名ではなく、デプロイメント名を使用してください。
