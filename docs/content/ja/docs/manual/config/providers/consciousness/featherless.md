title: Featherless.ai
description: AIRI での Featherless.ai チャット モデルの設定
---

Featherless.ai は、互換性のある API を介して AIRI のチャット モデルを提供します。

::: info Featherless.ai を選ぶ理由?
Featherless.ai でモデル アクセスをオープンしている場合は、その API キーを直接使用して AIRI を設定できます。
:::

## ステップ 1: API キーを取得する

1. [Featherless.ai](https://featherless.ai/)，在账户控制台创建 API Key.)を開いてログインします。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **設定 → サービス プロバイダー → チャット → Featherless.ai** を開き、**API キー**を入力します。デフォルトのベース URL は `https://api.featherless.ai/v1/` です。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、モデルを選択し、**設定 → 認識** に移動して有効にします。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウントのステータス、ネットワーク接続を確認してください。モデル リストの読み込みに失敗した場合は、ベース URL がデフォルト値のままであることを確認するか、Featherless.ai によって提供された正確なモデル ID を [認識] ページに入力します。
