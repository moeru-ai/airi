---
title: CometAPI
description: AIRI で CometAPI チャット モデルを設定する
---

CometAPI は AIRI でチャット モデルを提供し、独立した TTS および STT サービス プロバイダー ページもあります。

::: info CometAPI を選ぶ理由
同じ CometAPI アカウントでチャット、音声合成、音声認識を設定する場合は、これを選択できます。
:::

## ステップ 1: API キーを取得する

1. [CometAPI Console]を開いてログインします(https://www.cometapi.com/console/token)，创建 API Key.

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **設定 → サービス プロバイダー → チャット → CometAPI** を開き、**API キー**を入力します。デフォルトのベース URL は `https://api.cometapi.com/v1/` です。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、モデルを選択し、**設定 → 認識** に移動して有効にします。

## トラブルシューティング

Ping API が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストの読み込みに失敗した場合は、ベース URL がデフォルト値のままであることを確認するか、CometAPI によって提供される正確なモデル ID を [Awareness] ページに入力します。
