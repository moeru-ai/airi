Moonshot は、OpenAI 形式と互換性のあるチャット API を提供します。このページでの設定を完了すると、AIRI は Moonshot が提供するモデルを Awareness で使用できるようになります。

::: info ムーンショットを選ぶ理由
AIRI で Moonshot モデルを使用する場合、またはすでに Moonshot API キーを持っている場合は、このサービス プロバイダーを直接選択できます。
:::

## ステップ 1: API キーを取得する

1. [ムーンショットコンソール](https://platform.moonshot.cn/)。)を開く
2. [API キー] ページで新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害されたら、すぐにそれを取り消して、Moonshot コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [プロバイダー] → [チャット] → [ムーンショット]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.moonshot.cn/v1` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、Moonshot によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
