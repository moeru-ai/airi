---

LM Studio はモデルをネイティブに実行でき、ネイティブ API を提供します。自分のデバイスでモデルを実行したいユーザーに適しています。デフォルトでは API キーは必要ありません。

::: info LM Studioを選ぶ理由?
モデルをローカルで実行し、モデル ファイルを自分で管理したい場合は、クラウド API キーに依存しない LM Studio がオプションです。
:::

## ステップ 1: ローカル サービスを開始する

1. [LM Studioダウンロードページ](https://lmstudio.ai/download)安装并打开 LM Studio)からチャットモデルをダウンロードして読み込みます。
2. **ローカル サーバー** を開き、ローカル サーバーを起動します。
3. AIRI がローカル サービスにアクセスできない場合は、LM Studio のサーバー設定で CORS を有効にします。

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [LM Studio]** を開きます。
2. デフォルトのベース URL `http://localhost:1234/v1/` をそのまま使用します。
3. LM Studio サービスで認証が必要な場合は、API キーを入力します。それ以外の場合は空白のままにします。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、AIRI がローカル サービスに接続できるかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックしてロードされたモデルを選択します。

## トラブルシューティング

接続できない場合は、まずローカル サーバーが実行中であり、ポートがベース URL と一致していることを確認します。 AIRI と LM Studio が同じデバイス上にない場合は、AIRI デバイスからアクセスできる LAN アドレスを使用し、信頼されたネットワーク上でのみサービスを開きます。
