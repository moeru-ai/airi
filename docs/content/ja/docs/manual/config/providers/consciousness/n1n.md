n1n は、AIRI で API 互換のチャット モデル アクセスを提供します。

::: info n1n を選ぶ理由
n1n のモデル サービスを使用する場合は、そのサービス アドレスとアカウント資格情報を AIRI に入力できます。
:::

## ステップ 1: サービスへのアクセス方法を準備する

1. [n1n](https://n1n.ai/)，确认你的服务地址及是否需要 API キー)を開いてログインします。

::: warning 認証情報のセキュリティ
API キーはオプションですが、プライベート サービス アドレス、アクセス トークン、またはゲートウェイ構成を公開しないでください。
:::

## ステップ 2: AIRI で設定する

1. **「設定」→「サービスプロバイダー」→「チャット」→「n1n」**を開きます。デフォルトのベース URL は `https://api.n1n.ai/v1/` です。
2. n1n の現在のサービス要件に従って API キーを入力します。デプロイメントで匿名アクセスが許可されている場合は、デプロイ担当者の指示に従って空白のままにしてください。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワーク、サービス アドレス、資格情報が正しいかどうかをテストします。
2. **モデルの選択**: テストが成功したら、モデルを選択し、**設定 → 意識** に移動して有効にします。

## トラブルシューティング

検証に失敗した場合は、サービス アドレス、API キー、デプロイヤーのアクセス ポリシーを確認してください。サービスで匿名アクセスが許可されている場合は、デプロイヤーの指示に従って API キーを空白のままにし、AIRI を実行しているデバイスからアドレスにアクセスできることを確認します。
