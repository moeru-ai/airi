---
title: 火山エンジンのコーディング計画
description: AIRI での Volcano Engine コーディング プラン チャット モデルの設定
---

Volcano Engine コーディング プランは、AIRI の独立したプロバイダー カードとして利用できます。

::: info Volcano Engine コーディング プランを選択する理由?
アカウントで Volcano Engine コーディング プランを使用している場合は、対応するサービス プランと一致するようにこのカードを選択する必要があります。
:::

## ステップ 1: コーディング プランの認証情報を準備する

1. [Volcano Engine Console](https://console.volcengine.com/)，在コーディングプラン対応ページ)を開いてログインし、API Key、エンドポイント、モデル情報を取得します。

::: warning API キーのセキュリティ
API キーやエンドポイント認証情報をリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Volcengine コーディング プラン]** を開き、Volcengine コーディング プランに従ってプランの API キー、エンドポイント、およびモデル情報を入力します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークと資格情報が正しいかどうかをテストします。
2. **モデルの選択**: テストが成功したら、**設定 → 認識**で利用可能なモデルを選択します。

## トラブルシューティング

検証に失敗した場合は、API キー、エンドポイント、モデル情報が同じ Volcano Engine コーディング プランに属しているかどうかを確認します。モデルをロードできない場合は、プランがターゲット モデルへのアクセスを許可していることを確認してください。
