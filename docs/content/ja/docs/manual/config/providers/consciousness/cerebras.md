title: Cerebras
description: AIRI での Cerebras チャット モデルの設定
---

Cerebras は、互換性のある API を通じて AIRI のチャット モデルを提供します。

::: info なぜセレブラなのか？
すでに Cerebras API を使用していて、AIRI でアカウント利用可能なモデルを呼び出したい場合は、これを選択できます。
:::

## ステップ 1: API キーを取得する

1. [Cerebras Cloud](https://cloud.cerebras.ai/)，创建 API Key.)を開いてログインします。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Cerebras]** を開き、**API キー**を入力します。デフォルトのベース URL は `https://api.cerebras.ai/v1/` です。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、モデルを選択し、**設定 → 認識** に移動して有効にします。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウントのステータス、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、ベース URL がデフォルト値のままであることを確認するか、Cerebras が提供する正確なモデル ID を [認識] ページに入力します。
