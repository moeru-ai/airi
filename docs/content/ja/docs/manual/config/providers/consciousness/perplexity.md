Perplexity は、OpenAI 形式と互換性のあるチャット API を提供します。このページでの設定を完了すると、AIRI は Perplexity in Affairs によって提供されるモデルを使用できるようになります。

::: info Perplexity を選択する理由
すでに Perplexity API アカウントをお持ちで、その利用可能なモデルを AIRI で使用したい場合は、このプロバイダーを選択できます。
:::

## ステップ 1: API キーを取得する

1. [Perplexity API設定](https://www.perplexity.ai/settings/api)。)を開く
2. [API キー] ページで新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害されたら、すぐにそれを取り消し、Perplexity コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Perplexity]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.perplexity.ai` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、Perplexity によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
