Together AI は、OpenAI 形式と互換性のあるチャット API を提供します。このページでの設定を完了すると、AIRI は Together AI inwareness が提供するモデルを使用できるようになります。

::: info Together AI を選ぶ理由?
Together AI でモデルをデプロイまたは使用している場合は、対応する API キーを直接再利用できます。
:::

## ステップ 1: API キーを取得する

1. [Together AI API Key](https://api.together.ai/settings/api-keys)。)を開く
2. 新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害された場合は、直ちにそのキーを取り消し、Togetter AI コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Togetter AI] ** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.together.xyz/v1` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストの読み込みに失敗した場合は、Togetter AI によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
