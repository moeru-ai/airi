Groq は、OpenAI 形式と互換性のあるチャット API を提供します。このページでの設定を完了すると、AIRI は、Groq が提供するモデルを Awareness で使用できるようになります。

::: info Groq を選ぶ理由?
会話の応答性を重視し、ターゲット モデルが Groq で利用できる場合は、このプロバイダーを試してください。
:::

## ステップ 1: API キーを取得する

1. [Groqコンソール](https://console.groq.com/)。)を開く
2. [API キー] ページで新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害されたら、すぐにそれを取り消し、Groq コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Groq] ** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.groq.com/openai/v1` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、Groq が提供する正確なモデル ID を [認識] ページに手動で入力できます。
