Fireworks AI は、OpenAI 形式と互換性のあるチャット API を提供します。このページでの設定を完了すると、AIRI は Fireworks AI が提供するモデルをwarewareで使用できるようになります。

::: info Fireworks AI を選ぶ理由
すでに Fireworks AI でモデルまたは推論サービスを管理している場合は、同じ API 認証情報のセットを直接再利用できます。
:::

## ステップ 1: API キーを取得する

1. [Fireworks AI API キー](https://fireworks.ai/account/api-keys) を開いてログインし、必要な API キーまたは認証情報を作成します。
2. 新しい API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害された場合は、直ちにそのキーを取り消し、Fireworks AI コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **「設定」→「サービスプロバイダー」→「チャット」→「Fireworks AI」** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.fireworks.ai/inference/v1` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストの読み込みに失敗した場合は、Fireworks AI によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
