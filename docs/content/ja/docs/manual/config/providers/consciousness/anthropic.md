---
title: Anthropic
description: AIRI での Anthropic Claude チャット モデルの設定
---

Anthropic プロバイダーにより、AIRI はクロード チャット モデルを使用できるようになります。 AIRI は Anthropic の API アドレスと API キーを使用します。モデル リストは AIRI に組み込まれているため、開始するためにベース URL やモデル ID を手動で入力する必要はありません。

::: info Anthropic を選択する理由
すでに Claude API を使用している場合、または AIRI で Claude モデルを使用したい場合は、Anthropic を直接選択できます。
:::

## ステップ 1: API キーを作成する

1. [Anthropic Console](https://platform.claude.com/settings/keys) を開いてログインし、必要な API キーまたは認証情報を作成します。
2. キーの適切な名前、有効期間、割り当て制限を設定します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害されたら、すぐにそれを取り消して、Anthropic コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [Anthropic]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://api.anthropic.com/v1/` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

Ping API はネットワーク接続をチェックし、短いチャット リクエストを送信します。失敗した場合は、API キーが利用可能であること、アカウントに制限があること、ネットワークが Anthropic API にアクセスできるかどうかを確認してください。

モデル セレクターに予期したモデルが表示されない場合は、まず AIRI を更新するか、サービス プロバイダーから提供された正確なモデル ID を [認識] ページで手動で入力します。
