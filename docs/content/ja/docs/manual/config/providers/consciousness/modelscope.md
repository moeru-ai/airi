---
title: ModelScope
description: AIRI での ModelScope チャット モデルの設定
---

ModelScope は、推論 API を通じて AIRI にチャット モデルを提供します。

::: info ModelScope を選ぶ理由
ModelScope でモデルへのアクセスを管理している場合は、このカードを使用して、対応する API キーを直接入力します。
:::

## ステップ 1: API キーを取得する

1. [ModelScope](https://modelscope.cn/)，在账户控制台创建 API Key.)を開いてログインします。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [ModelScope]** を開き、**API キー**を入力します。デフォルトのベース URL は `https://api-inference.modelscope.cn/v1/` です。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、モデルを選択し、**設定 → 意識** に移動して有効にします。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウントのステータス、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、ベース URL がデフォルト値のままであることを確認するか、ModelScope が提供する正確なモデル ID を [認識] ページに入力します。
