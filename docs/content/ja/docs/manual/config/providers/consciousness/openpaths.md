OpenPaths は、OpenAI 形式と互換性のあるチャット API サービスプロバイダーです。このページでの設定を完了すると、AIRI は OpenPaths が提供するチャット モデルを「認識」して使用できるようになります。

::: info なぜ OpenPath なのか?
OpenPaths API キーをすでに持っている場合、または OpenPaths API キーが提供するモデルを使用したい場合は、このサービス プロバイダーを直接選択できます。
:::

## ステップ 1: API キーを取得する

1. [OpenPaths](https://openpaths.io/))にアクセスし、アカウントを登録し、API Keyを作成します。
2. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害された場合は、直ちにそのキーを取り消し、OpenPaths コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [OpenPaths]** を開きます。
2. API キーを基本設定に貼り付けます。
3. デフォルトのベース URL `https://openpaths.io/v1` をそのまま使用します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、ここをクリックして使用する特定のモデルを選択します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデル リストのロードに失敗した場合は、OpenPaths によって提供される正確なモデル ID を [認識] ページに手動で入力できます。
