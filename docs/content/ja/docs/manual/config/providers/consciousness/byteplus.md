BytePlus は、AIRI で Ark 互換のチャット サービス構成を使用します。

::: Info BytePlusを選ぶ理由?
BytePlus で Ark モデルのエンドポイントを作成した場合は、そのエンドポイントと認証情報を AIRI で直接使用できます。
:::

## ステップ 1: BytePlus 認証情報を準備する

1. [BytePlus コンソール](https://console.byteplus.com/) を開いてログインし、必要な API キーまたは認証情報を作成します。

::: warning API キーのセキュリティ
API キーやエンドポイント認証情報をリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [チャット] → [BytePlus]** を開き、BytePlus コンソールに従って API キー、エンドポイント、またはモデル情報を入力します。
2. エンドポイント ID を推測しないでください。コンソールで作成されたエンドポイントの実際の ID をコピーします。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークと資格情報が正しいかどうかをテストします。
2. **モデルの選択**: テストが成功したら、**設定 → 意識** に移動して、サービス プロバイダーとモデルを選択します。

## トラブルシューティング

検証が失敗した場合は、API キー、エンドポイント ID、およびモデル情報が同じ BytePlus Ark プロジェクトからのものであるかどうかを確認します。エンドポイント ID を手動で推測しないでください。実際の ID をコンソールからコピーします。
