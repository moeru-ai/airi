---
title: こころ（ローカルTTS）
description: AIRI でネイティブ Kokoro 音声合成を設定する
---

Kokoro は AIRI のネイティブ モデルとして音声合成を実行します。

::: info なぜココロを選ぶのか？
音声コンテンツをローカルで処理する必要があり、デバイスがモデルの実行条件を満たしている場合は、Kokoro を選択できます。
:::

## ステップ 1: ローカルの動作環境を準備する

1. WebGPU をサポートする環境で AIRI を開きます。初めて使用する場合は、モデルのダウンロードが完了するまで待ちます。
2. このオプションにはクラウド API キーは必要ありませんが、ローカルのダウンロード スペース、メモリ、およびコンピューティング リソースが使用されます。

::: warning ローカルリソースの使用状況を警告する
ローカル モデルは、ダウンロード スペース、メモリ、およびコンピューティング リソースを占有します。デバイスのリソースが不足している場合は、強制的に有効化しないでください。
:::

## ステップ 2: AIRI で設定する

1. **「設定」→「サービスプロバイダー」→「音声合成」→「ココロ」**を開きます。
2. AIRI が提供する利用可能な Kokoro モデルを選択します。

## ステップ 3: 構成を確認する

1. **モデルとトーンの選択**: モデルを準備したら、トーンを選択し、**設定 → サウンド** に移動して有効にします。
2. 聞く短いテキストを入力します。正常に再生できれば、モデルの準備が完了したことを意味します。

## トラブルシューティング

モデルをロードできない場合は、ブラウザが WebGPU をサポートしているかどうか、デバイスのリソースが十分であるかどうかを確認し、ページを再度開いてダウンロードが完了するまで待ちます。
