---
title: MiniMax Speech（TTS）
description: AIRI での MiniMax 音声合成の設定
---

MiniMax Speech は、AIRI に音声合成モデルとプリセットサウンドを提供します。

::: info MiniMax Speech を選択する理由
すでに MiniMax を使用していて、その中国語または英語のプリセット サウンドを直接使用したい場合は、それを選択できます。
:::

## ステップ 1: API キーを取得する

1. [MiniMax Open Platform]を開いてログインします(https://platform.minimaxi.com/)，开通 API使用権。
2. APIキー管理ページでキーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [MiniMax Speech]** を開き、API キーを貼り付けます。
2. サービスプロバイダーが別のアドレスを明示的に提供しない限り、デフォルトのサービスアドレス `https://api.minimax.io` を維持します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルとサウンドの選択**: テストが成功したら、`speech-2.8-hd` や `speech-2.8-turbo` などの利用可能なモデルとサウンドを選択し、**設定 → サウンド** に移動して有効にします。
3. 短いテキストを入力して聞いて、正常に再生できることを確認します。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。モデルまたはサウンドが利用できない場合は、MiniMax アカウントで現在開いているリストが優先されます。
