---
title: 火山エンジン (TTS)
description: AIRI での Volcano Engine 音声合成の設定
---

Volcano エンジン音声合成では、API キーとアプリケーション情報を AIRI に入力する必要があります。

::: info ボルケーノ エンジンを選ぶ理由
Volcano Engine で音声アプリケーションを作成し、サウンド リソースを管理している場合は、この設定セットを AIRI で再利用できます。
:::

## ステップ 1: アプリケーションの資格情報を準備する

1. [Volcano Engine Console](https://console.volcengine.com/)，创建或打开语音应用。)を開いてログインします。
2. アプリケーションの **アプリ ID** をコピーし、対応する **API キー**を作成します。
3. 両方の情報が同じアカウントとアプリケーション構成からのものであることを確認します。

::: 警告 API キーのセキュリティ
API キーまたはアプリ ID に対応する認証情報を開示しないでください。漏洩後は、サービス プロバイダー コンソールのキーを直ちに交換してください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [火山エンジン]** を開きます。
2. API キーとアプリ ID を入力します。互換性のあるゲートウェイを使用しない限り、ベース URL ではインターフェイスのデフォルト値が使用されます。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワーク、API キー、およびアプリ ID が正しく入力されているかどうかをテストします。
2. **モデルとトーンの選択**: テストが成功したら、トーンを選択し、**設定 → サウンド** に移動して有効にします。
3. 短いテキストを入力して聞いて、正常に再生できることを確認します。

## トラブルシューティング

検証に失敗した場合は、アプリ ID と API キーが同じアプリケーションのものであるかどうかを確認してください。音が出ない場合は、アプリで音声合成が有効になっていてトーンが選択されていることを確認してください。
