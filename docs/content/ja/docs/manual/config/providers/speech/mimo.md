MiMo は、プリセット音色、サウンド デザイン、サウンド クローンの 3 つの音声合成モードをサポートしています。

::: info Xiaomi MiMo を選ぶ理由?
中国語のサウンドをプリセットする必要がある場合、またはテキストの説明を使用してサウンドをデザインしたい場合は、MiMo を選択できます。
:::

## ステップ 1: API キーを取得する

1. [Xiaomi MiMo Platform](https://platform.xiaomimimo.com/)，确认账户已开通 API使用権)を開いてログインします。
2. API キーを作成し、コピーして安全な場所に保管します。

::: warning 音サンプルとAPI Keyセキュリティ
サウンドのクローン作成には、Base64 データ URI 形式のオーディオ サンプルが必要です。使用許可のあるサウンドのみをアップロードしてください。 API キーや他の人のサウンド サンプルを公開しないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [Xiaomi MiMo]** に API キーを入力します。
2. サービスプロバイダーが別のアドレスを提供しない限り、デフォルトのベース URL: `https://api.xiaomimimo.com/v1/` を維持します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルとサウンドの選択**: テストが成功したら、プリセット サウンド、サウンド デザイン、またはサウンド クローン モデルを選択し、**設定 → サウンド制作** で有効にします。
3. 聞く短いテキストを入力します。サウンドデザインにはスタイルの説明が必要です。サウンドのクローン作成には、正規のサウンド サンプルも必要です。

## トラブルシューティング

リクエストが失敗した場合は、API キーとモデルの選択を確認してください。サウンドのクローン作成が失敗した場合は、サンプルが有効な Base64 データ URI であること、およびサウンドを使用する権限があることを確認してください。
