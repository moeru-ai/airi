---
title: CometAPI（TTS）
description: AIRI で CometAPI 音声合成を設定する
---

CometAPI は、互換性のあるインターフェイスを通じて音声合成を提供します。

::: info CometAPI を選ぶ理由
すでに CometAPI を使用してモデルと認証情報を管理している場合は、API キーを AIRI で直接再利用できます。
:::

## ステップ 1: API キーを取得する

1. [CometAPI Console]を開いてログインします(https://www.cometapi.com/console/token)，创建 API Key.
2. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [CometAPI Speech]** を開き、API キーを入力します。
2. デフォルトのベース URL: `https://api.cometapi.com/v1/` をそのまま使用します。プロキシまたは互換性のあるゲートウェイを使用する場合にのみ変更してください。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルとサウンドの選択**: テストが成功したら、利用可能なモデルとサウンドを選択し、**設定 → サウンド** に移動して有効にします。
3. 短いテキストを入力して聞いて、正常に再生できることを確認します。

## トラブルシューティング

検証に失敗した場合は、API キー、アカウント残高、ネットワーク接続を確認してください。モデル リストが空の場合は、アカウントが現在音声モデルにアクセスできることを確認します。
