イレブンラボは AIRI の応答を音声に合成します。

::: info イレブンラボを選ぶ理由?
イレブンラボ アカウントのサウンドを使用し、利用可能なサウンドを AIRI で直接選択する場合は、このプロバイダーを選択します。
:::

## ステップ 1: API キーを取得する

1. [イレブンラボAPIキー設定](https://elevenlabs.io/app/settings/api-keys)，创建密钥。)を開いてログインします。
2. キーに識別しやすい名前と適切な使用制限を付けます。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。リークが疑われる場合は、直ちに取り消して、イレブンラボ コンソールで再作成してください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [イレブンラボ]** を開きます。
2. API キーを基本設定に貼り付けます。
3. インターフェースのデフォルトのベース URL をそのまま使用します。互換性のある独自のゲートウェイを使用する場合にのみ変更してください。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルとサウンドの選択**: テストが成功したら、**設定 → サウンド**で使用するモデルとサウンドを選択します。
3. 短いテキストを入力し、それを聞きます。音声が正常に再生できれば、設定は成功です。

## トラブルシューティング

API への ping が失敗した場合は、API キー、アカウント制限、ネットワーク接続を確認してください。機種は表示されるのに音が鳴らない場合は、「音声」で有効な機種と音が選択されているか確認してください。
