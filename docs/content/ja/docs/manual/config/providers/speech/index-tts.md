---
title: インデックス TTS (ローカル TTS)
description: AIRI のローカル Index-TTS サービスへの接続
---

Index-TTS は、ネイティブ HTTP サービスを介して AIRI と連携する音声合成オプションです。

::: info Index-TTS を選ぶ理由
既に Index-TTS をローカルで実行していて、サウンド データをローカル ネットワーク上に残しておきたい場合は、これを選択できます。
:::

## ステップ 1: ローカル サービスを開始する

1. Index-TTS の展開手順に従ってサービスを開始します。
2. AIRI はデフォルトで `http://localhost:11996/tts/` に接続します。他のホストまたはポートを使用する場合は、完全なアドレスを記録してください。
3. サービスがモデルと音色の情報を返せることを確認します。

::: ローカルサービスのセキュリティを警告する
ローカル サービス ポートを信頼できないパブリック ネットワークに公開しないでください。リスニング アドレスを変更する場合は、アクセス制御の責任があることを確認してください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [Index-TTS by Bilibili]** を開きます。
2. ローカル サービスと一致するベース URL を入力します。

## ステップ 3: 構成を確認する

1. プロバイダー設定でモデルと利用可能な音声を選択します。
2. 同じページのプレイグラウンドで短いテキストを入力し、音声が再生されることを確認します。

## トラブルシューティング

接続できない場合は、サービスが実行中であること、ベース URL に正しいポートが含まれていることを確認し、ローカル ファイアウォールまたはリバース プロキシを確認してください。トーン リストが空の場合は、サービスの `audio/voices` インターフェイスが利用可能かどうかを確認します。
