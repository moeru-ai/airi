---
title: Deepgram（TTS）
description: AIRI でのディープグラム音声合成の設定
---

Deepgram は、AIRI で音声合成モデルの Aura ファミリを提供します。

::: info ディープグラムを選ぶ理由
すでに Deepgram を使用している場合、または Aura の音声範囲から選択したい場合は、この統合を使用できます。
:::

## ステップ 1: API キーを取得する

1. [Deepgram Console](https://console.deepgram.com/) を開いてログインし、必要な API キーまたは認証情報を作成します。
2. プロジェクトに音声合成の使用権限があることを確認します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [ディープグラム]** を開き、API キーを基本設定に貼り付けます。
2. インターフェースのデフォルトのベース URL をそのまま使用します。互換性のあるゲートウェイを自分で導入する場合にのみ変更してください。

## ステップ 3: 構成を確認する

1. プロバイダー設定でモデルと利用可能な音声を選択します。
2. 同じページのプレイグラウンドで短いテキストを入力し、音声が再生されることを確認します。

## トラブルシューティング

プレイグラウンドでリクエストが完了しない場合は、プロジェクトの API キー、アカウントの権限、ネットワーク接続を確認してください。トーン リストが空の場合は、モデルを選択する前に認証情報を再テストしてください。
