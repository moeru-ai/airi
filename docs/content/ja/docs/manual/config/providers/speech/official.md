---
title: AIRI公式音声合成（TTS）
description: AIRI で公式に提供されている音声合成を使用する
---

公式音声合成では AIRI のログインステータスが使用されるため、サードパーティの API キーを別途入力する必要はありません。

::: info AIRI公式音声合成を選ぶ理由
すでに公式の AIRI プロバイダーを使用していて、サードパーティの認証情報の構成を減らしたい場合は、まずこのオプションを試してください。
:::

## ステップ 1: アカウントにログインします

1. AIRI アカウントを使用してログインを完了します。公式音声サービスは現在のログイン状態によって異なります。
2. サードパーティ API キーを作成したり入力したりする必要はありません。

::: warning アカウントと制限
公式サービスの利用可能なモデル、割り当て、およびリージョンはサーバーによって決定されます。アカウントのログインステータスやブラウザセッション情報を共有しないでください。
:::

## ステップ 2: AIRI で設定する

1. **「設定」→「サービスプロバイダー」→「音声合成」→「AIRI公式音声合成」**を開きます。
2. AIRI が現在利用可能なモデルとサウンドをサーバーからロードするまで待ちます。

## ステップ 3: 構成を確認する

1. **[設定] → [音声]** でモデルとサウンドを選択し、短いテキストを使用してそれを聞いてください。
2. 正常に再生できれば、設定は成功したことになります。サーバーがストリーミング音声を提供していない場合、ストリーミング オプションは表示されません。

## トラブルシューティング

モデル リストをロードできない場合は、アカウントがログインしていること、ネットワークが正常であること、および利用可能なクレジットがあることを確認してください。デスクトップでリチャージできない場合は、AIRI Web バージョンの **[設定] → [Flux]** に移動してリチャージを完了してください。
