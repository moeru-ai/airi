---
title: ブラウザネイティブ音声合成 (TTS)
description: AIRI Web でのブラウザネイティブ音声合成の使用
---

ブラウザネイティブの音声合成では、AIRI のローカル モデル機能を使用し、クラウド API キーを必要としません。

::: info ブラウザネイティブの音声合成を選択する理由
Web バージョンを使用していて、サードパーティの音声プロバイダーへのテキストの送信を最小限に抑えたい場合は、このオプションを試すことができます。
:::

## ステップ1：ブラウザ環境の確認

1. AIRI Web バージョンを使用します。このサービス プロバイダーはデスクトップには表示されません。
2. このサービス プロバイダー カードが表示される前に、ブラウザが WebGPU をサポートしているか、デバイスのメモリが 8 GB 以上であることを確認してください。

::: warning 装置の互換性
この機能はブラウザとハードウェアの条件によって異なります。サービス プロバイダー カードが表示されない場合、またはモデルを実行できない場合は、代わりにクラウド TTS またはデスクトップ ローカル ソリューションを使用してください。
:::

## ステップ 2: AIRI で設定する

1. Web バージョンで [設定] → [サービス プロバイダー] → [音声合成] → [ブラウザ (ローカル)] ** を開きます。
2. モデルが準備されるのを待って、モデルと音色を選択します。

## ステップ 3: 構成を確認する

1. **[設定] → [音声]** でサービスプロバイダー、モデル、およびサウンドを選択します。
2. 短いテキストを入力して聞きます。正常に再生できれば設定は成功です。

## トラブルシューティング

カードが表示されない場合は、現在のバージョンが Web バージョンであるかどうか、デバイスが WebGPU またはメモリ要件を満たしているかどうかを確認してください。モデルを実行できない場合は、代わりにクラウド TTS またはデスクトップ ローカル ソリューションを使用してください。
