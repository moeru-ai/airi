---

ブラウザ（ローカル）音声合成はまだ利用できません。AIRI の設定ページは現在 Work in Progress のプレースホルダーを表示するため、モデルと音声を準備または選択できません。

::: warning 利用不可
このプロバイダーはまだ動作する設定には使用できません。現在の利用状況を明確にするためにのみ記載しています。
:::

## 利用可能な代替手段

AIRI で実装済みのローカル音声合成には [Kokoro](./kokoro.md) を使用するか、対応するクラウド TTS プロバイダーを選択してください。
