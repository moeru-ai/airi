---
title: Google Gemini（TTS）
description: AIRI での Google Gemini 音声音声合成の設定
---

Google Gemini Audio 音声合成は、Gemini 認証情報とオーディオ出力をサポートするモデルを使用します。

::: info Google Gemini を選ぶ理由
AIRI で Google Gemini を設定していて、同じサービス プロバイダーで音声出力機能を使用したい場合は、このオプションを選択できます。
:::

## ステップ 1: API キーを取得する

1. [Google AI Studio](https://aistudio.google.com/app/apikey)，创建 API Key.)を開いてログインします。
2. アカウントが音声出力をサポートする Gemini モデルを使用できることを確認します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
Gemini API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [Google Gemini]** に Gemini API キーを入力します。
2. エンタープライズ ゲートウェイまたは互換性のあるプロキシを使用している場合を除き、インターフェイスのデフォルトのベース URL を保持します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルとトーンの選択**: テストが成功したら、インターフェイスにリストされている音声出力をサポートするモデルを選択し、**設定 → 音声** に移動して有効にします。
3. 短いテキストを入力して聞いて、正常に再生できることを確認します。

## トラブルシューティング

検証が失敗した場合は、API キー、アカウントのリージョンの可用性、およびネットワーク接続を確認してください。リクエストは成功したが音が出ない場合は、選択したモデルが音声出力をサポートしていることを確認してください。
