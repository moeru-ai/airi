title: Microsoft Azure Speech（TTS）
description: AIRI での Microsoft Azure 音声合成の設定
---

Microsoft Azure Speech は、AIRI で Azure 音声合成機能を提供します。

::: info Microsoft Azure Speech を選択する理由
チームが既に Azure で音声リソースとリージョン構成を管理している場合は、同じ資格情報を使用する方が便利です。
:::

## ステップ 1: Azure Speech リソースを準備する

1. [Azure Portal](https://portal.azure.com/) を開いてログインし、必要な API キーまたは認証情報を作成します。
2. **API キー**とリソースのリージョンを記録します。両方とも同じ音声リソースからのものである必要があります。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
Azure キーを使用すると、音声リソースへのアクセスが可能になります。送信したり、スクリーンショットを撮ったり、共有したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [Microsoft/Azure Speech]** を開きます。
2. API キーと地域情報を入力します。互換性のあるゲートウェイを使用しない限り、ベース URL はインターフェイスのデフォルトのままです。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワーク、API キー、およびリージョンが正しく入力されているかどうかをテストします。
2. **モデルとトーンの選択**: テストが成功したら、トーンを選択し、**設定 → サウンド**で有効にします。
3. 短いテキストを入力して聞いて、正常に再生できることを確認します。

## トラブルシューティング

検証が失敗した場合は、領域が Speech リソースと一致しているかどうかを確認することが優先されます。音声がない場合は、「音声」で音声が選択されていることを確認し、リソースに使用可能なクレジットがあるかどうかを確認してください。
