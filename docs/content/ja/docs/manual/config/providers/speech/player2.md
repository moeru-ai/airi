---
title: Player2 Speech（TTS）
description: AIRI のローカル Player2 Speech サービスに接続する
---

Player2 Speech は、ローカル サービス経由で接続される音声合成オプションです。

::: info Player2 Speech を選択する理由
Player2 Speech をローカルまたは信頼できる LAN 上ですでに実行している場合は、AIRI をサービスに接続して、サービスが提供するサウンドを読み取ることができます。
:::

## ステップ 1: ローカル サービスを開始する

1. Player2 Speech サービスを開始し、ヘルスチェックが利用可能であることを確認します。
2. AIRI はデフォルトで `http://localhost:4315/v1/` に接続します。サービスが別のアドレスで実行されている場合は、完全なベース URL を記録してください。

::: ローカルサービスのセキュリティを警告する
ローカル サービス ポートを信頼できないパブリック ネットワークに公開しないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [Player2 音声]** を開きます。
2. サービス アドレスと一致するベース URL を入力します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、AIRI がサービスに接続できるかどうかをテストします。
2. **モデルとサウンドの選択**: テストが成功したら、`player2-tts` とサービスから返されたサウンドを選択し、**設定 → サウンド** に移動して有効にします。
3. 短いテキストを入力して聞いて、正常に再生できることを確認します。

## トラブルシューティング

接続に失敗した場合は、サービスの `/health` 応答とベース URL を確認してください。トーン リストが空の場合は、サービスの `/tts/voices` インターフェイスにアクセスできることを確認します。
