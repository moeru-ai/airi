---
title: OpenAI互換API（TTS）
description: AIRI での音声合成プロバイダーとしての OpenAI または OpenAI 互換 API の設定
---

このページは、OpenAI または OpenAI 互換の音声インターフェイスを提供するサービス プロバイダーを構成するために使用されます。設定が完了したら、「Voice」でモデルと音色を選択します。

::: info TTS と互換性のある OpenAI を選択する理由
音声サービス プロバイダーが OpenAI 互換の音声合成インターフェイスを明示的に提供している場合は、同じ設定方法を使用して AIRI にアクセスできます。 `/v1` で終わる API アドレスまたは `sk-` で始まるキーを持っているだけでは、サービスの互換性は保証されません。
:::

## ステップ 1: API キーを取得する

1. 選択したサービスプロバイダーの管理コンソールにログインします。
2. API キーまたは開発者設定ページで API キーを作成します。
3. キーをコピーし、安全な場所に保管します。

::: 警告 API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを含めたり、他の人に送信したりしないでください。キーが侵害された場合は、すぐにそれを取り消し、プロバイダー コンソールで新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声合成] → [OpenAI 互換 API]** を開きます。
2. 使用する API キーと TTS モデル ID を入力します。
3. OpenAI 公式サービスを使用する場合は、デフォルトのベース URL: `https://api.openai.com/v1/` を維持します。互換性のあるサービスを使用する場合は、サービス プロバイダーのドキュメントで提供される API ルート アドレスを入力します。
4. 必要に応じて、話す速度を調整します。

## ステップ 3: 構成を確認する

1. 「音声」で設定したサービスプロバイダー、モデル、トーンを選択します。
2. テストテキストを入力し、「テスト」をクリックします。
3. 音声が正常に再生できれば、設定は成功です。エラーが表示された場合は、エラーメッセージに従って認証情報、モデルID、ベースURLを確認してください。

## トラブルシューティング

音が出ない場合は、まずサービスプロバイダーがサポートするモデルと音色を選択しているかどうかを確認してください。互換性のあるサービスを使用する場合は、そのサービスが OpenAI 音声合成インターフェイスを明示的にサポートしていることを確認してください。
