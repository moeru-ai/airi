---
title: デスクトップローカル音声合成 (TTS)
description: AIRI デスクトップでのネイティブ音声合成の使用
---

デスクトップ ローカル音声合成は AIRI デスクトップ バージョンでのみ利用可能であり、クラウド API キーは必要ありません。

::: info デスクトップローカル音声合成を選択する理由?
これは、AIRI デスクトップを使用していて、クラウド TTS への依存を減らしたい場合のオプションです。
:::

## ステップ 1: デスクトップ環境を確認する

1. AIRI デスクトップ バージョンを使用します。このサービス プロバイダーは Web バージョンには表示されません。
2. デバイスにローカル モデルに十分なディスク容量とコンピューティング リソースがあることを確認します。

::: warning 動作環境
このサービスプロバイダーは Web バージョンには表示されません。モデルの実行にはローカル ディスクとコンピューティング リソースが使用されます。
:::

## ステップ 2: AIRI で設定する

1. AIRI デスクトップ バージョンで [設定] → [サービス プロバイダー] → [音声合成] → [アプリ (ローカル)] ** を開きます。
2. 利用可能なモデルとサウンドを選択します。準備ができたら、**設定 → 音声化** に移動して有効にします。

## ステップ 3: 構成を確認する

1. 短いテキストを入力し、それを聞きます。
2. 音声が正常に再生できれば設定成功です。

## トラブルシューティング

サービスプロバイダーが表示されない場合は、デスクトップバージョンを実行していることを確認してください。モデルの準備に失敗した場合は、ディスク容量とデバイスのリソースを確認して、再試行してください。
