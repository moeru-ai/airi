---
title: Xiaomi MiMo (ASR/STT)
description: AIRI での Xiaomi MiMo 音声認識の設定
---

MiMo は、ネイティブ音声理解モデルを使用して音声の文字起こしを完了します。

::: info Xiaomi MiMo を選ぶ理由?
すでに MiMo を使用している場合、またはそのマルチモーダル モデルを使用してオーディオ コンテンツを処理したい場合は、このサービス プロバイダーを選択できます。
:::

## ステップ 1: API キーを取得する

1. [Xiaomi MiMo Platform](https://platform.xiaomimimo.com/)，确认账户已开通 API使用権)を開いてログインします。
2. API キーを作成し、コピーして安全な場所に保管します。

::: warning API キーと音声データ
API キーを公開しないでください。クラウド文字起こしを使用すると、認識される音声がサービス プロバイダーに送信されます。最初にデータ処理要件を確認してください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [音声認識] → [Xiaomi MiMo]** に API キーを入力します。
2. サービスプロバイダーが別のアドレスを提供しない限り、デフォルトのベース URL: `https://api.xiaomimimo.com/v1/` を維持します。

## ステップ 3: 構成を確認する

1. **Ping API**: このボタンをクリックして、ネットワークが接続されているかどうか、また API キーが正しく入力されているかどうかをテストします。
2. **モデルの選択**: テストが成功したら、`mimo-v2-omni` またはインターフェイスにリストされている利用可能なモデルを選択します。次に、**設定 → 聴覚** に移動して有効にします。
3. マイクへのアクセスを許可し、短い音声入力を実行してテキストを出力できることを確認します。

## トラブルシューティング

リクエストが失敗した場合は、API キー、モデルの選択、ネットワーク接続を確認してください。テキスト結果がない場合は、AIRI がシステムのマイク許可を取得していることを確認します。
