---
title: AIRI公式音声認識（ASR/STT）
description: AIRI で公式のリアルタイム音声認識を使用する
---

公式音声認識では AIRI のログインステータスが使用されるため、サードパーティの API キーを別途入力する必要はありません。

::: info AIRI 公式音声認識を選択する理由
すでに公式の AIRI プロバイダーを使用していて、リアルタイム音声入力をすぐに有効にしたい場合は、まずこのオプションを試してください。
:::

## ステップ 1: アカウントにログインします

1. AIRI アカウントを使用してログインを完了します。公式のリアルタイム識別は、現在のログイン ステータスに依存します。
2. サードパーティ API キーを作成したり入力したりする必要はありません。

::: 警告アカウントと音声データ
リアルタイム認識により、音声が公式サービスに送信されます。機密情報を含むテスト音声を使用したり、アカウント セッション情報を共有したりしないでください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービスプロバイダー] → [音声認識] → [AIRI 公式音声認識]** を開きます。
2. `Auto` またはサーバーが提供するモデルを選択し、**設定 → 聴覚** に移動して有効にします。

## ステップ 3: 構成を確認する

1. AIRI にマイクの使用を許可し、短い音声入力を実行します。
2. 転写されたテキストが表示されれば、設定は成功です。

## トラブルシューティング

モデルが使用できない場合は、アカウントがログインしていること、ネットワークが正常であること、および利用可能なクレジットがあることを確認してください。テキスト結果がない場合は、システムのマイクの許可を確認してください。
