---
title: ブラウザ Web スピーチ API (ASR/STT)
description: AIRI Web でのブラウザ内音声認識の使用
---

Web Speech API はブラウザによって提供される音声認識機能を使用するため、API キー用の別のアプリケーションは必要ありません。

::: info Web Speech API を選択する理由
Web 上で音声入力をすぐに試しているだけで、ブラウザが Web Speech API をサポートしている場合、これが最小限の構成オプションです。
:::

## ステップ 1: ブラウザのサポートを確認する

1. AIRI Web バージョンを使用します。 Web Speech APIはデスクトップ版(Electron)では提供されません。
2. 現在のブラウザが Web Speech API をサポートしており、マイクのアクセス許可を許可する準備ができていることを確認します。

::: warning ブラウザ制限
Web Speech API はブラウザ環境でのみ利用可能であり、デスクトップ バージョンの AIRI (Electron) ではサポートされていません。認識機能はブラウザ、ネットワーク環境、言語によって異なる場合があります。
:::

## ステップ 2: AIRI で設定する

1. Web バージョンで [設定] → [サービス プロバイダー] → [音声認識] → [Web Speech API] ** を開きます。
2. 認識言語を選択し、必要に応じて継続的な認識と中間結果のオプションを選択します。

## ステップ 3: 構成を確認する

1. **[設定] → [聴覚]** に移動し、Web Speech API と音声入力デバイスを選択します。
2. ブラウザがマイクにアクセスできるようにし、短い音声入力テストを開始します。
3. 転写されたテキストが表示されれば、設定は成功です。

## トラブルシューティング

テキスト結果がない場合は、ブラウザのマイクの許可、選択した入力デバイス、認識されている言語を確認してください。ブラウザがこの API をサポートしていない場合は、代わりにローカルまたはクラウド ASR を使用してください。
