---
title: ブラウザネイティブ音声認識 (ASR/STT)
description: AIRI Web でのブラウザネイティブ音声認識の使用
---

ブラウザのローカル音声認識は、AIRI のローカル モデル機能を使用し、クラウド API キーを必要としません。

::: info ブラウザネイティブの音声認識を選択する理由
Web バージョンを使用していて、サードパーティプロバイダーへの音声の送信を最小限に抑えたい場合は、このオプションを試すことができます。
:::

## ステップ1：ブラウザ環境の確認

1. AIRI Web バージョンを使用します。このサービス プロバイダーはデスクトップには表示されません。
2. このサービス プロバイダー カードが表示される前に、ブラウザが WebGPU をサポートしているか、デバイスのメモリが 8 GB 以上であることを確認してください。

::: 警告装置の互換性
ローカル モデルはデバイス リソースを消費します。サービス プロバイダー カードが表示されない場合、または認識を開始できない場合は、代わりに Web Speech API、クラウド ASR、またはデスクトップ ローカル ソリューションを使用してください。
:::

## ステップ 2: AIRI で設定する

1. Web バージョンで [設定] → [サービス プロバイダー] → [音声認識] → [ブラウザ (ローカル)] ** を開きます。
2. モデルが準備されるまで待ち、モデルを選択して、**設定 → 聴覚** で有効にします。

## ステップ 3: 構成を確認する

1. ブラウザがマイクにアクセスして短い音声入力を実行できるようにします。
2. 転写されたテキストが表示されれば、設定は成功です。

## トラブルシューティング

カードが表示されない場合は、現在のバージョンが Web バージョンであるかどうか、デバイスが WebGPU またはメモリ要件を満たしているかどうかを確認してください。認識が開始できない場合は、ブラウザのマイクの許可を確認してください。
