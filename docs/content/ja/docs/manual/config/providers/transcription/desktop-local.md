---
title: デスクトップローカル音声認識 (ASR/STT)
description: AIRI デスクトップでのネイティブ音声認識の使用
---

デスクトップのローカル音声認識は AIRI デスクトップ バージョンでのみ利用可能であり、クラウド API キーは必要ありません。

::: info デスクトップのローカル音声認識を選択する理由
これは、AIRI デスクトップを使用していて、クラウドベースの文字起こしサービスへの依存を減らしたい場合のオプションです。
:::

## ステップ 1: デスクトップ環境を確認する

1. AIRI デスクトップ バージョンを使用します。このサービス プロバイダーは Web バージョンには表示されません。
2. デバイスにローカル モデルに十分なディスク容量とコンピューティング リソースがあることを確認します。

::: warning 動作環境
このサービスプロバイダーは Web バージョンには表示されません。モデルの実行にはローカル ディスクとコンピューティング リソースが使用されます。
:::

## ステップ 2: AIRI で設定する

1. AIRI デスクトップ バージョンで [設定] → [サービス プロバイダー] → [音声認識] → [アプリ (ローカル)] ** を開きます。
2. 利用可能なモデルを選択します。準備ができたら、**[設定] → [聴覚]** に移動して有効にします。

## ステップ 3: 構成を確認する

1. AIRI にマイクの使用を許可し、短い音声入力を実行します。
2. AIRI に認識結果が表示されれば、設定は成功です。

## トラブルシューティング

サービスプロバイダーが表示されない場合は、デスクトップバージョンを実行していることを確認してください。モデルの準備が失敗した場合は、ディスク容量とデバイス リソースを確認してください。テキスト結果がない場合は、システムのマイクの許可を確認してください。
