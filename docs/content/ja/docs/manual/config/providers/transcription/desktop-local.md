---
title: デスクトップローカル音声認識 (ASR/STT、利用不可)
description: デスクトップローカル音声認識はまだ設定できません
---

アプリ（ローカル）音声認識はまだ利用できません。AIRI の設定ページは現在 Work in Progress のプレースホルダーを表示するため、モデルを準備または有効化できません。

::: warning 利用不可
このプロバイダーはまだ動作する設定には使用できません。現在の利用状況を明確にするためにのみ記載しています。
:::

## 利用可能な代替手段

対応するブラウザでは [Web Speech API](./web-speech-api.md) を使用するか、対応するクラウド文字起こしプロバイダーを選択してください。
