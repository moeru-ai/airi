---
title: ナノバナナ（芸術作品）
description: AIRI での Nano Banana イメージ生成の設定
---

Nano Banana は、Google AI Studio API キーを使用して画像を生成します。設定が完了したら、**設定 → Artistry** でサービス プロバイダーを選択できます。

::: info ナノバナナを選ぶ理由
すでに Google AI Studio API キーをお持ちで、AIRI の組み込み Gemini 画像モデルと解像度オプションを直接使用したい場合は、それを選択できます。
:::

## ステップ 1: API キーを取得する

1. [Google AI Studio API キー](https://aistudio.google.com/app/apikey)，创建 API キー)を開いてログインします。
2. 選択したイメージ モデルがアカウントと地域で利用できることを確認します。
3. キーをコピーし、安全な場所に保管します。

::: warning API キーのセキュリティ
API キーをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。キーが侵害された場合は、すぐにキーを取り消して、Google AI Studio で新しいキーを作成します。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [アート] → [Nano Banana]** を開き、API キーを貼り付けます。
2. デフォルトのモデル: `gemini-3.1-flash-image-preview`、`gemini-3-pro-image-preview`、または `gemini-2.5-flash-image` を選択します。
3. デフォルトの解像度を 1K、2K、または 4K から選択します。

## ステップ 3: 構成を確認する

1. **[設定] → [Artistry]** を開き、**Nano Banana** を選択します。
2. 機密情報を含まないプロンプトワードを使用して画像を生成します。
3. 画像が正常に返されたということは、API キー、モデル、解像度の構成が利用可能であることを意味します。

## トラブルシューティング

認証に失敗した場合は、API Keyが有効かどうかを確認してください。モデルが利用できない場合、またはリクエストが拒否された場合は、Google AI Studio アカウント、利用可能なリージョン、現在のモデルのステータスを確認してください。ビルドが失敗した場合は、1K 解像度または別の利用可能なモデルに切り替えて、再試行してください。
