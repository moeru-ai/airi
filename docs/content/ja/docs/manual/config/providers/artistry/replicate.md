---
title: 複製 (芸術的創造)
description: AIRI でのレプリケート イメージ生成の設定
---

Replicate を使用すると、AIRI はクラウド内の画像を使用してモデルを生成できます。構成が完了したら、**設定 → Artistry** で画像生成ソースとして複製を選択できます。

::: info レプリケートを選択する理由
イメージ モデルを自分でデプロイするのではなく、Replicate で利用可能なモデルから選択したい場合は、クラウド推論サービスを選択できます。
:::

## ステップ 1: API トークンを取得する

1. [API トークンの複製](https://replicate.com/account/api-tokens) を開いてログインし、必要な API キーまたは認証情報を作成します。
2. アカウントに利用可能な請求方法または制限が設定されていることを確認します。
3. トークンをコピーし、適切に保存します。

::: warning API トークンのセキュリティ
トークンをリポジトリに送信したり、スクリーンショットを撮ったり、他の人に送信したりしないでください。漏洩後は、すぐにトークンを取り消して、レプリケート コンソールで新しいトークンを作成してください。
:::

## ステップ 2: AIRI で設定する

1. **[設定] → [サービス プロバイダー] → [アート] → [複製]** を開きます。
2. API トークンを貼り付けます。
3. デフォルトのモデル ID を入力します。 AIRI のデフォルト値は `black-forest-labs/flux-schnell` です。モデルの複製ページに表示される正確な ID を参照してください。
4. 必要に応じて、デフォルトの画面比率 (デフォルト `16:9`) と推論ステップ数 (デフォルト 4) を設定します。

## ステップ 3: 構成を確認する

1. **[設定] → [Artistry]** を開き、**[複製]** を選択します。
2. 機密情報を含まないプロンプトワードを使用して画像を生成します。
3. イメージが正常に返されたということは、トークン、モデル ID、およびアカウント クォータが使用可能であることを意味します。

## トラブルシューティング

認証に失敗した場合は、Tokenが完全に貼り付けられているか確認してください。リクエストが拒否または失敗した場合は、アカウント制限、モデルのアクセス権限、およびモデル ID を確認してください。生成された結果が期待を満たさない場合は、まずモデルがサポートするスケールとパラメーターの範囲を確認し、推論ステップ数を減らすか、モデルを置き換えます。
