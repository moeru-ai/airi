## AIRI へようこそ!

初めて AIRI を使用する場合は、AIRI に「頭脳」を与える必要があります。 「開始」をクリックしてモデル構成ページに入ります。

### サービスソースの選択

ここで大規模モデルのプロバイダーを選択してください。 OpenAI、DeepSeek、Ollama などを選択することも、OpenAI 形式と互換性のある他のすべての API サイトを使用することもできます。
API キーを入力します。 OpenAI 互換 API を選択した場合は、大規模モデル プロバイダーの API エンドポイントであるベース URL も設定する必要があります。ここでは、シリコンベースのフローを例として使用します。

![](/assets/screenshot-api-example.avif)

### モデルを選択してください

AIRIのBrainのモデル一覧からお好みのAIモデルを選択してください。

::: tip

think 関数を備えたモデルを使用している場合、応答の生成に時間がかかることがあります。会話の流暢さを向上させるには、非思考モデルを使用することをお勧めします。

:::

### 最初の会話を始めます

ダイアログボックスにテキストを入力して AIRI~ と会話します
![](/assets/screenshot-chat.avif)

## 目、耳、口

テキスト対話に加えて、AIRI には多くのインタラクティブなメソッドもあります。ページ右上の[設定]→[ボディモジュール]をクリックすると、彼女にインタラクティブな機能を追加できます。

### AIRIに話してもらいましょう

［音声］をクリックして音声合成を設定します。組み込みの Kokoro TTS は中国語をサポートしていないため、[+] をクリックしてソースを追加する必要があります。入力後、「音声」ページに戻り、新しく追加した音声合成プロバイダーを選択します。

### AIRIに音を聞かせてください

文字入力やチャットが面倒な場合があります。この場合、音声からテキストへの変換機能を設定する必要があります。 [聴覚]をクリックしてテキスト変換を設定します。ブラウザに付属の STT は中国語のサポートが不完全であり、これを実現するには新しいソースを作成する必要があります。
Speech-to-Text プロバイダーを構成した後、使用可能なマイクを選択する必要もあります。 AIRI ホームページのダイアログ ボックスでマイク アイコンをクリックしてマイクを選択し、テキストに切り替えることができます。

## キャラクターカード

AIRIには「ReLU」というキャラクターカードが内蔵されており、モデルに自分でキャラクターカードを書くこともできます。ホームページ右上にキャラクターカード切り替えボタンがあるか、設定からも切り替えられます。

### キャラクターカードの内容

キャラクターカードにはAIRIの名前、説明、性格、行動などが含まれており、AIを高度にカスタマイズできます。

### キャラクターカードの設定

各キャラクター カードは機体モジュールごとに特定のモデル プロバイダーを選択でき、キャラクター カードを切り替えることでモデルをすばやく切り替えることができます。
