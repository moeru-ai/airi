## Welcome to AIRI!

When using AIRI for the first time, you need to give AIRI a "brain". Click "Start" to enter the model configuration page

### Select service source

Choose your large model provider here. You can choose OpenAI, DeepSeek, Ollama, etc., or you can use all other API sites that are compatible with the OpenAI format.
Fill in your API key. If you choose OpenAI compatible API, you also need to configure the Base URL, which is the API endpoint of the large model provider. Here, silicon-based flow is used as an example.

![](/assets/screenshot-api-example.avif)

### Select model

Select an AI model you like from the list of models for AIRI’s Brain.

::: tip

If you're using a model with a think function, generating responses can take a long time. It is recommended to use non-thinking models to improve conversational fluency.

:::

### Start your first conversation

Enter text in the dialog box to talk to AIRI~
![](/assets/screenshot-chat.avif)

## Eyes, ears and mouth

In addition to text dialogue, AIRI also has many interactive methods. Click [Settings]->[Body Module] in the upper right corner of the page to add more interactive functions to her.

### Let AIRI speak

Click [Voice] to configure speech synthesis. The built-in Kokoro TTS does not support Chinese, so we need to click [+] to add a source. After filling in the relevant content, return to the [Voice] page and select the newly added speech synthesis provider.

### Let AIRI hear the sound

Sometimes typing and chatting is too troublesome. In this case, you need to configure a voice-to-text function. Click [Hearing] to configure text conversion. The STT that comes with the browser has incomplete support for Chinese, and we still need to create a new source to achieve this.
After configuring the speech-to-text provider, you also need to select an available microphone. You can click the microphone icon in the AIRI homepage dialog box to select the microphone and the switch to text.

## Character Card

AIRI has a built-in character card called "ReLU", and you can also write your own character card for the model. You can find the button to switch character cards in the upper right corner of the homepage, or you can switch from the settings.

### Character card content

The character card contains AIRI's name, description, personality, behavior, etc. You can highly customize your AI.

### Character card settings

Each character card can select a specific model provider for each airframe module, and you can quickly switch your models by switching character cards.
