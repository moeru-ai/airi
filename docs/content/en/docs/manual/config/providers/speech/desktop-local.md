---
title: Desktop local speech synthesis (TTS)
description: Using native speech synthesis in AIRI desktop
---

Desktop local speech synthesis is only available in AIRI desktop version and does not require a cloud API key.

::: info Why choose desktop local speech synthesis?
This is an option if you use AIRI desktop and want to reduce your reliance on cloud TTS.
:::

## Step 1: Confirm the desktop environment

1. Use the AIRI desktop version; this service provider will not appear in the web version.
2. Make sure the device has sufficient disk space and computing resources for the local model.

::: warning operating environment
This service provider will not appear in the web version. Model runs use local disk and computing resources.
:::

## Step 2: Configure in AIRI

1. Open **Settings → Service Provider → Speech Synthesis → App (Local)** in the AIRI desktop version.
2. Select available models and sounds; when ready, go to **Settings → Vocalization** to enable them.

## Step 3: Verify configuration

1. Enter a short text and listen to it.
2. If the voice can be played normally, the configuration is successful.

## Troubleshooting

When the service provider does not appear, make sure you are running the desktop version. When model preparation fails, check the disk space and device resources and try again.
