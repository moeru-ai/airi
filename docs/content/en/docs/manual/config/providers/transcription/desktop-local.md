---
title: Desktop local speech recognition (ASR/STT)
description: Using native speech recognition in AIRI desktop
---

Desktop local speech recognition is only available in the AIRI desktop version and does not require a cloud API Key.

::: info Why choose desktop local speech recognition?
This is an option if you use AIRI desktop and want to reduce your reliance on cloud-based transcription services.
:::

## Step 1: Confirm the desktop environment

1. Use the AIRI desktop version; this service provider will not appear in the web version.
2. Make sure the device has sufficient disk space and computing resources for the local model.

::: warning operating environment
This service provider will not appear in the web version. Model runs use local disk and computing resources.
:::

## Step 2: Configure in AIRI

1. Open **Settings → Service Provider → Speech Recognition → App (Local)** in the AIRI desktop version.
2. Select an available model; when ready, go to **Settings → Hearing** to enable it.

## Step 3: Verify configuration

1. Allow AIRI to use the microphone and perform a short voice input.
2. If the recognition result can be displayed in AIRI, it means the configuration is successful.

## Troubleshooting

When the service provider does not appear, make sure you are running the desktop version. When model preparation fails, check disk space and device resources; when there are no text results, check system microphone permissions.
