# AI 翻唱

AIRI 的 AI 翻唱模块允许将任意歌曲转换为指定角色的声音。

## 功能概述

- **角色翻唱**：使用预训练的 RVC 模型将歌曲转换为 Airi 的声音
- **零样本翻唱**：上传参考音频，无需训练即可克隆声音（Seed-VC）
- **自动流水线**：人声分离 → 音高提取 → 音色转换 → 混音，全自动完成
- **产物追踪**：每次任务生成完整的 manifest.json 审计记录

## 技术栈

| 阶段 | 技术 |
|------|------|
| 人声分离 | MelBand-RoFormer / BS-RoFormer |
| 音高提取 | RMVPE |
| 音色转换 | RVC（主线）/ Seed-VC（零样本） |
| 混音处理 | FFmpeg |
