---
title: 异星工厂
description: 将 AIRI 连接到受信任的 Factorio 服务器
---

Factorio 集成让 AIRI 使用游戏服务器地址、端口和玩家名称连接外部游戏服务。AIRI 桌面端提供连接配置；你仍需要自行准备可访问的 Factorio 服务器和与之配套的服务端集成。

## 前提条件

* 可访问的 Factorio 服务器。
* 服务器管理员已允许你的账号和所使用的服务端集成连接。
* 服务器地址、端口和游戏内用户名。

::: warning 仅连接受信任的服务器
该集成会让 AIRI 与游戏服务器交换上下文和操作请求。不要将它用于不受信任的公共服务器，也不要在公共聊天、截图或 Issue 中公开服务器地址、令牌或账号信息。
:::

## 在 AIRI 中配置

1. 打开 **设置 → 机体模块 → 异星工厂（Factorio）**。
2. 启用 Factorio 集成。
3. 填写服务器地址、端口和你的游戏内用户名；默认端口为 `34197`。
4. 点击 **保存**。页面显示“已配置”仅代表三个字段已填写；是否能实际连接仍取决于服务器和服务端集成。

## 排查

* 检查服务器地址和端口能否从运行 AIRI 的设备访问。
* 确认防火墙、VPN 和服务器白名单没有阻止连接。
* 确认用户名与服务器中的玩家名称一致。
* 如果配置已保存但仍不能交互，检查服务端集成的日志；AIRI 桌面端不随附一个可直接部署的 Factorio 机器人服务。
