# 大语言模型 (LLM) 配置

LLM 是 AIRI 的“大脑”。配置正确后，她才能理解你的话并做出回应。

### 1. 快速上手：聚合服务商 (推荐)

如果你希望在一个地方管理所有模型，且不想处理复杂的网络问题，推荐使用：

#### **OpenRouter / 302.ai**
- **优势**：国内直连友好，支持 GPT-4o, Claude 3.5 等顶级模型。
- **配置步骤**：
  1. 在 [服务商官网] 获取 API Key。
  2. 在 AIRI 设置中选择 `OpenAI 兼容模式`。
  3. **Base URL** 填写：`https://api.302.ai/v1` (确保包含 /v1)。
  4. **API Key** 填入你获取的卡密。
