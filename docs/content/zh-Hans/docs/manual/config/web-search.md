---
title: 配置网络搜索
description: 使用 Tavily 让 AIRI 在对话中搜索最新信息
---

网络搜索让 AIRI 在需要最新资料或不熟悉的信息时查询互联网。它使用你自己的 Tavily API Key；启用后，AIRI 会在合适的对话中自行调用搜索工具，并在回答中附上实际使用的来源链接。

## 前提条件

- 已安装并启动 AIRI。
- 已拥有 Tavily 账号，并从 [Tavily](https://tavily.com/) 控制台创建 API Key。
- 已配置可使用工具调用的聊天服务商和模型。若 AIRI 无法使用工具，请先更换为支持工具调用的模型。

::: warning API Key 安全
Tavily API Key 只应保存在当前设备。不要提交到仓库、发送给他人，或放入角色卡、日志和截图中。若怀疑密钥已泄露，请立即在 Tavily 控制台撤销它并创建新密钥。
:::

## 配置步骤

1. 打开 **设置 → 机体模块 → 网络搜索**。
2. 开启「启用网络搜索」。
3. 在「Tavily API 密钥」中粘贴 API Key。
4. 出现「网络搜索已就绪」后即可返回聊天；设置会自动保存，无需另点保存按钮。

关闭开关或清空 API Key 后，AIRI 不会再向 Tavily 发送搜索请求。

## AIRI 何时会搜索

AIRI 优先使用已有知识；当你明确要求搜索，或问题涉及会快速变化的信息时，它才会使用网络搜索。比如新闻、价格、最近发布的版本、当前活动、实时榜单或最新文档。

若希望它搜索得更准确，请直接说清楚目标与范围，例如：

- “搜索 AIRI 最新稳定版的发行说明，并附上链接。”
- “查找 Tavily 官方文档中有关 API Key 的说明。”
- “只搜索 `github.com/moeru-ai/airi` 上最近一周的更新。”

搜索结果会包含来源链接。AIRI 只能引用实际查询到的链接；如果回答没有找到足够的资料，应继续搜索或明确说明不确定之处。

## 隐私、可靠性与安全

每次搜索会将查询文字发送至 Tavily。因此不要在搜索词中包含 API Key、密码、访问令牌、私人地址或其他不应提供给第三方的信息。搜索结果也可能包含错误、过期或带有偏见的内容。

::: warning 请核实重要信息
搜索结果仅供 AIRI 参考，不会自动改变你原本的提问或操作目标。涉及账户、安全、医疗、法律或财务的内容，请打开来源链接自行核实，并优先参考官方或一手来源。
:::

## 常见问题

### 显示已配置，但 AIRI 没有搜索

先确认网络搜索开关仍处于开启状态，并确认当前聊天模型支持工具调用。接着直接在聊天中要求“搜索并附上来源链接”；如果仍未调用，请检查模型服务商是否允许工具调用请求。

### 提示 API Key、权限或额度错误

回到 Tavily 控制台确认密钥完整、仍有效，并检查账户的可用额度或访问权限。复制时不要带入前后的空格或换行；更换密钥后返回 AIRI 重新粘贴即可。

### 搜索结果不准确或不够新

在提问中说明时间范围、地点和希望使用的来源，例如“只查过去一周”或“仅使用官方文档”。对重要结论打开所附链接核对；网络搜索不能替代专业建议或独立判断。
