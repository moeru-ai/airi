# AI服务API申请教程

本文档提供了国内外主流AI服务的API申请流程指南，帮助您快速获取API密钥并配置到AIRI中。

## 国内AI服务

### Kimi (Moonshot AI)

Kimi是由月之暗面开发的AI服务，提供中文友好的大语言模型。

#### 申请步骤
1. **访问官网**：前往 [Kimi开发者平台](https://platform.moonshot.cn/)
2. **注册账号**：使用手机号注册并完成实名认证
3. **创建API密钥**：
   - 登录后进入「控制台」
   - 点击「API密钥」→「创建密钥」
   - 为密钥命名并保存
4. **获取API地址**：默认API地址为 `https://api.moonshot.cn/v1/`

#### 注意事项
- 新用户有免费额度
- API密钥需要妥善保管，不要分享给他人
- 如有网络问题，可以尝试使用国内镜像地址

### Doubao (ByteDance)

Doubao是字节跳动开发的AI服务，提供多种模型选择。

#### 申请步骤
1. **访问官网**：前往 [字节跳动云服务](https://cloud.volcengine.com/)
2. **注册账号**：使用手机号注册并完成企业认证（个人也可）
3. **开通服务**：
   - 搜索「方舟大模型」
   - 点击「立即开通」
4. **创建API密钥**：
   - 进入「API密钥管理」
   - 点击「新建密钥」
5. **获取API地址**：默认API地址为 `https://ark.cn-beijing.volces.com/api/v3/`

#### 注意事项
- 需要先开通服务才能创建API密钥
- 有免费调用额度，超出后需付费

### 阿里云通义千问

通义千问是阿里云开发的AI服务。

#### 申请步骤
1. **访问官网**：前往 [阿里云AI平台](https://ai.aliyun.com/)
2. **注册账号**：使用手机号注册并完成认证
3. **开通服务**：
   - 搜索「通义千问」
   - 点击「立即开通」
4. **创建API密钥**：
   - 进入「AccessKey管理」
   - 点击「创建AccessKey」
5. **获取API地址**：根据不同模型获取对应的API地址

## 国外AI服务

### OpenAI

OpenAI提供GPT系列模型，是最流行的AI服务之一。

#### 申请步骤
1. **访问官网**：前往 [OpenAI平台](https://platform.openai.com/)
2. **注册账号**：使用邮箱注册，可能需要国外手机号验证
3. **创建API密钥**：
   - 登录后进入「API Keys」
   - 点击「Create new secret key」
   - 复制并保存密钥
4. **获取API地址**：默认API地址为 `https://api.openai.com/v1/`

#### 注意事项
- 需要国外手机号验证
- 新用户有$5-$18的免费额度
- 可能需要科学上网才能访问

### Anthropic (Claude)

Anthropic提供Claude系列模型，以安全性和准确性著称。

#### 申请步骤
1. **访问官网**：前往 [Anthropic平台](https://console.anthropic.com/)
2. **注册账号**：使用邮箱注册
3. **创建API密钥**：
   - 登录后进入「API Keys」
   - 点击「Create Key」
   - 为密钥命名并保存
4. **获取API地址**：默认API地址为 `https://api.anthropic.com/v1/`

#### 注意事项
- 需要国外手机号验证
- 新用户有一定的免费额度
- 可能需要科学上网才能访问

## API配置指南

### 在AIRI中配置API
1. **打开设置**：在AIRI应用中点击「设置」→「AI服务」
2. **选择服务**：从下拉菜单中选择您要配置的AI服务
3. **填写信息**：
   - API密钥：粘贴您获取的API密钥
   - API地址：填写对应的API地址（通常系统会自动填充）
4. **测试连接**：点击「测试连接」按钮验证配置是否正确
5. **保存设置**：点击「保存」按钮

### 常见问题

#### API密钥无效
- 检查是否正确复制了完整的API密钥
- 确认API密钥是否已过期
- 验证账号是否有足够的额度

#### 连接失败
- 检查网络连接是否正常
- 对于国外服务，确认是否使用了科学上网
- 验证API地址是否正确

#### 模型不可用
- 检查账号是否有权限访问该模型
- 确认模型是否已在平台上启用
- 查看平台文档了解模型的使用限制

## 最佳实践

1. **保护API密钥**：不要在代码中硬编码API密钥，使用环境变量或配置文件
2. **监控使用量**：定期查看API使用情况，避免超出预算
3. **设置合理的超时**：为API调用设置合理的超时时间，提高用户体验
4. **错误处理**：实现完善的错误处理机制，提高应用的稳定性
5. **使用国内服务**：在国内网络环境下，优先使用国内AI服务以获得更好的响应速度

## 故障排除

如果遇到API相关问题，可以尝试以下步骤：

1. **检查网络连接**：确保网络连接稳定
2. **验证API密钥**：确认API密钥是否正确且未过期
3. **检查API地址**：验证API地址是否正确
4. **查看平台状态**：访问服务提供商的状态页面，确认服务是否正常
5. **联系客服**：如果问题持续存在，联系服务提供商的客服寻求帮助
